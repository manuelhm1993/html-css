    5. CSS básico - Enlazar CSS
        5.1. Estilos en línea: se agregan como atributos a las etiquedas html para modificar sus propiedades
        <p style="color: red; font-size: 20px;">Contenido</p> (es una mala práctica)
        5.2. Estilos internos: se crea una etiqueta <style></style> dentro del <head></head> y allí se
        colocan las propiedades apuntando a las etiquedas con selectores (no recomendado)
        5.3. Estilos externos: crear un archivo independiente con extensión .css y escribir todas las reglas
        de estilo allí, luego enlazar en el html a través de la etiqueta link dentro del head (recomendado)
        <link rel="stylesheet" href="css/app.css">
        5.4. Estructura de una regla css:
            5.4.1. Selector: p
            5.4.2. Bloque de declaraciones: {}
            5.4.3. Declaración: propiedad: valor;
            5.4.4. Regla css: p { color: red; }
    6. CSS básico - Selectores
        6.1. Son la forma que se tiene para seleccionar uno o varios elementos dentro de un documento html
        6.2. Selector de etiqueta o elemento: h1 {}
        6.3. Selector de clase: se crea una clase en css (.destacado) y se escriben allí las reglas, luego
        se agrega el atributo class="" a los elementos html que se desea modificar <img class="destacado">
        6.4. Selector de id: es el que tiene más prioridad, pero solo se aplica a 1 elemento, se agrega el
        atributo id a una etiqueta html <section id="main"></section> y se llama en css con #main
    7. CSS básico - Propiedades y fuentes de texto

        7.5. line-height: indica el interlineado
        7.6. text-transform: transforma el texto entre minúscula, mayúscula, capitalize...

    9. CSS básico - Modelo de caja
        9.1. Es un concepto que define como se posicionan y diseñan los elementos html en una página web
        9.2. Cada elemento es una caja, esta caja incluye varias propiedades que definen como se muestra
        el elemento en la página
            9.2.1. Caja:
                - Content: es el núcleo del box model, es el área donde se muestra el texto
                - Padding: es el espacio entre el contenido y el borde del elemento, aumenta el área al 
                rededor del contenido dentro del elemento, pero bajo el borde
                - Border: es una línea que rodea al contenido y al padding, se puede controlar el estilo,
                el grosor y el color del borde
                - Margin: es el espacio al rededor de la caja que está fuera del borde, este espacio separa
                el elemento de otros elementos en la página y a diferencia de las demás, no es parte de la 
                caja
        9.3. Width y Height
            9.3.1. Definen el ancho y el alto de un elemento, normalmente se mide en pixeles, pero para
            aplicar el responsive, se usan porcentajes, en ese caso se usa un contenedor padre y al hijo
            se le aplica el porcentaje que usará del espacio del padre 
        9.4. Padding: espacio que hay entre el contenido y el borde de una caja o contenedor, aumenta el
        tamaño que hay al rededor del contenido, pero adentro de los bordes, básicamente relleno
            9.4.1. padding: arriba, derecha, abajo, izquierda
            9.4.2. padding: vertical, horizontal
            9.4.3. padding-(top,right,bottom,left)
        9.5. Margin: espacio que hay entre el contenedor o caja y el resto de elementos, se define igual
        que el padding, pero se debe tener en consideración el margin-collapse, significa que el margen superior
        o de más tamaño será el margen final respecto a su siguiente contenedor, esto evita cálculos matemáticos
            9.5.1. Centrar una caja: margin: 0 auto;
        9.6. Border
            9.6.1. border-width: permite modificar el grosor o tamaño del borde
            9.6.2. border-style: permite modificar el estilo del borde
            9.6.3. border-color: permite modificar el color del borde
            9.6.4. Shorthand: border: medida, estilo, color
            9.6.5. border-(top, right, bottom, left): permiten modificar lados individuales del borde
            9.6.6. border-radius: permite modificar el redondeo del borde, se puede trabajar en px o en %
            el estilo de redondeo cambia de uno a otro
        9.7. Tamaño de caja (box-sizing)
            9.7.1. El box-sizing es una propiedad de css que se usa para afectar el funcionamiento del modelo 
            de caja predeterminado, el modelo de caja es como los navegadores calculan el total del tamaño de un
            elemento
            9.7.2. box-sizing: content-box; es el valor por defecto que tienen las cajas, el tamaño que se le da
            a la caja solo se aplica al contenido, es decir, width y height se aplican al contenido de la caja,
            pero si se agregan padding, border y margin, esto se suma al tamaño de la caja
            9.7.3. box-sizing: border-box; esto es una novedad y aplica el tamaño al borde de la caja, es decir,
            que a width y height se les suma padding y border y eso es el tamaño de la caja, no el margin. Si
            luego se agregan padding y border, entonces el contenido se ajusta para que no sobrepase la medida

    11. CSS básico - Colores
        11.1. Existen muchas formas de trabajar con colores, pero las tres más comunes son:
            11.1.1. RGB || RGBA: rojo, verte y azul | rojo, verde y azul alpha, los tres colores primarios de la luz y
            un modificador de opacidad. Tiene 256 posibles combinaciones de colores para cada valor
            11.1.2. HSL || HSLA: Trabaja con una circunferencia para medir los colores, el porcentaje de saturación y el
            porcentaje de iluminación, donde el 50% de iluminación es la base
            11.1.3. Hexadecimal || Hexadecima - Alpha: permite trabajar con códigos hexadecimales para tener más control
            en base a 16, también se trabaja con RGB, pero cada letra va en el rango: #R0-F, G0-F, B0-F, se puede usar 
            la nomenclatura completa #000000 o el método abreviado #000, para ambos casos se le agrega el alpha, otra capa
            que controla la opacidad de 0-F
    12. CSS básico - Unidades de medida
        12.1. Medidas absolutas: se usan para tener control y ser precisos, es decir, tamaños fijos, no es la mejor solución para el responsive
            12.1.1. Pixeles px: son medidas absolutas, son la parte más mínima de área de luz, se usan para definir normalmente alto y ancho de las imágenes, márgenes, padding y el tamaño de las tipografías
            12.1.2. Puntos pt: son medidas absolutas que se usan para definir el tamaño de impresión
            12.1.3. Milímetros mm, centímetros cm y pulgadas in: en las pantallas varía el diseño, pero en impresión si se respetan las medidas, en otras palabras, son medidas físicas y absolutas
        12.2. Medidas relativas: son unidades que dependen de otro valor, por ejemplo otro contenedor, el tamaño fuente, etc... son ideales para el responsive design
            12.2.1. Porcentaje %: siempre va a a ser respecto al tamaño de su contenedor, funciona con margin y padding, pero en textos se toma el font-size del contenedor padre
            12.2.2. em: va a tomar el valor que se le indique y lo va a multiplicar por el font-size del elemento padre, es decir, si el padre tiene font-size de 30px, 1em del hijo vale 30px. Si el contenedor padre no tiene definido font-size, entonces busca a un padre por arriba
            12.2.3. rem: es la misma fórmula del em, pero en lugar de tomar de referencia al contenedor padre, va a tomar de referencia al contenedor raíz, es decir, la etiqueta html, si no es definido font-size, lo define el navegador, normalmente son 16px
        12.3. Medidas del viewport: se usan para trabajar las medidas de la pantalla y su unidad es v
