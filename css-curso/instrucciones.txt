    5. CSS básico - Enlazar CSS
        5.1. Estilos en línea: se agregan como atributos a las etiquedas html para modificar sus propiedades
        <p style="color: red; font-size: 20px;">Contenido</p> (es una mala práctica)
        5.2. Estilos internos: se crea una etiqueta <style></style> dentro del <head></head> y allí se
        colocan las propiedades apuntando a las etiquedas con selectores (no recomendado)
        5.3. Estilos externos: crear un archivo independiente con extensión .css y escribir todas las reglas
        de estilo allí, luego enlazar en el html a través de la etiqueta link dentro del head (recomendado)
        <link rel="stylesheet" href="css/app.css">
        5.4. Estructura de una regla css:
            5.4.1. Selector: p
            5.4.2. Bloque de declaraciones: {}
            5.4.3. Declaración: propiedad: valor;
            5.4.4. Regla css: p { color: red; }
    6. CSS básico - Selectores
        6.1. Son la forma que se tiene para seleccionar uno o varios elementos dentro de un documento html
        6.2. Selector de etiqueta o elemento: h1 {}
        6.3. Selector de clase: se crea una clase en css (.destacado) y se escriben allí las reglas, luego
        se agrega el atributo class="" a los elementos html que se desea modificar <img class="destacado">
        6.4. Selector de id: es el que tiene más prioridad, pero solo se aplica a 1 elemento, se agrega el
        atributo id a una etiqueta html <section id="main"></section> y se llama en css con #main
    7. CSS básico - Propiedades y fuentes de texto
        7.1. Son las propiedades para formatear texto, la mayoría son básicas e intuitivas, como: color,
        fuente, tamaño, pero a destacar está el grosor:
            7.1.1. Las tipografías varían en grosor de 100-900 de fino a grueso
            7.1.2. bold = 700
            7.1.3. bolder detecta el máximo grosor y lo aumenta al siguiente nivel
        7.2. font-style: indica si va de normal o cursiva 
        7.3. text-align: indica el justificado
        7.4. text-decoration: indica el subrayado
        7.5. line-height: indica el interlineado
        7.6. text-transform: transforma el texto entre minúscula, mayúscula, capitalize...
    8. CSS básico - tipografías externas
        8.1. Recurso: https://fonts.google.com/
        8.2. Básicamente si se quiere usar una fuente muy especial que no está instalada en todos los SO
        se busca en google fonts y se puede instalar via CDN que es lo más sencillo, se busca el link y
        se coloca en el head de html, luego se crea la regla css
        8.3. La forma recomendada es descargando la fuente e instalarla en el proyecto, el mismo proceso
        de buscar en google fonts, pero en este caso en lugar de usar el link, se debe descargar la fuente
        agregarla a los assets y luego configurarla con la regla @font-face
            8.3.1. Regla:               @font-face {
            8.3.2. Nombre de la fuente:     font-family: "Nombre";
            8.3.3. Ubicación y formato:     src: url("./relative.ttf") format("truetype");
            8.3.4. Grosor:                  font-weight: 100-900; 
                                        }
            Luego se crea la regla css llamando al nombre y el grosor
    9. CSS básico - Modelo de caja
        9.1. Es un concepto que define como se posicionan y diseñan los elementos html en una página web
        9.2. Cada elemento es una caja, esta caja incluye varias propiedades que definen como se muestra
        el elemento en la página
            9.2.1. Caja:
                - Content: es el núcleo del box model, es el área donde se muestra el texto
                - Padding: es el espacio entre el contenido y el borde del elemento, aumenta el área al 
                rededor del contenido dentro del elemento, pero bajo el borde
                - Border: es una línea que rodea al contenido y al padding, se puede controlar el estilo,
                el grosor y el color del borde
                - Margin: es el espacio al rededor de la caja que está fuera del borde, este espacio separa
                el elemento de otros elementos en la página y a diferencia de las demás, no es parte de la 
                caja
        9.3. Width y Height
            9.3.1. Definen el ancho y el alto de un elemento, normalmente se mide en pixeles, pero para
            aplicar el responsive, se usan porcentajes, en ese caso se usa un contenedor padre y al hijo
            se le aplica el porcentaje que usará del espacio del padre

        9.5. Padding
